CSS

  • border-radius: adjusts the border (%)

  • Background-color: sets the background color of the cell

  • Border: sets a border for the cell

  • Text-align: aligns the text to whichever direction

  • Color: color of the font

  • Font-size: size of the font

  • Margin: "area" of the cell

  • Padding: The area of the cell that is reserved for it only

  • Width: adjusts the width of the cell

  • Height: adjusts the height of the cell

  • list-style: sets the style of the bullet points

  • Display: how the text is displayed
